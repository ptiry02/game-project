# Road Crossing Game

This is a simple 2D game made with html css and javascript.

try to get to the other side without beeing run over! 😄

---

## Gameplay:

Use the arrow keys <kbd>↑</kbd> <kbd>↓</kbd> <kbd>←</kbd> <kbd>→</kbd> to move.

Complete the 3 levels.
